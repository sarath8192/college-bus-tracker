# Kubernetes Basics

This document explains the Kubernetes setup for the College Bus Tracking System.

## Goal

The goal of this phase is to run the frontend and backend Docker containers using Kubernetes locally.

## Kubernetes Components Used

- Namespace
- Deployment
- Service
- Secret
- NodePort
- Liveness Probe
- Readiness Probe
